# Changelog

Все значительные изменения в Starvell Cardinal будут документироваться в этом файле.

## [0.0.7] - 2026-01-07

### Добавлено

- **Plugins**: Поддержка регистрации Telegram команд из плагинов через `COMMANDS`
- **Plugins**: Поддержка callback-хэндлеров через `CALLBACKS`
- **Plugins**: Поддержка текстовых хэндлеров через `TEXT_HANDLERS`

### Улучшено

- **Plugins**: Автоматическая регистрация команд плагинов при загрузке через aiogram router


### Документация

- **Plugins API**: Обновлена документация по плагинам — изменены названия переменных метаданных (`PLUGIN_NAME` → `NAME`, `PLUGIN_VERSION` → `VERSION` и т.д.)
