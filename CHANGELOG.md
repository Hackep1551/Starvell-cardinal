# Changelog

Все значительные изменения в Starvell Cardinal будут документироваться в этом файле.

## [0.0.14] - 2026-01-07

### Улучшено

- **Черный список**: Теперь работает по ID пользователей вместо username
  - Проверка ЧС теперь в уведомлениях сообщений (полностью игнорирует)
  - Проверка ЧС в автоответах на заказы и отзывы
  - Более надежная идентификация пользователей
  
- **Уведомления сообщений**: Теперь показывают ID отправителя вместо nickname
  - Более точная идентификация пользователей
  - Упрощенное копирование ID для добавления в ЧС

- **Уведомления о поднятии лотов**: Теперь отображаются при успешном поднятии
  - Настройка уже была в конфиге (lotBump), теперь работает корректно
  - Показывает ID игры и время с последнего поднятия

- **Оптимизация логирования**: Значительно снижена многословность логов
  - Логи планировщика APScheduler теперь ERROR вместо WARNING (только WARNING в DEBUG режиме)
  - Убраны избыточные логи запуска сервисов (автообновление, keep alive)
  - Авто-поднятие лотов: подробные логи только при первой проверке и обнаружении новых категорий
  - Все еще логируется факт успешного поднятия лотов
